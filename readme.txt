Allows pop-up tooltips to be shown for links to locations in OotiniMaps.

== Description ==

OotiniMaps is an atlas of locations from Star Wars: The Old Republic (SWTOR). This plugin allows links to OotiniMaps locations to generate a pop-up showing a tooltip with the location's map. Any waypoints included in the link are also shown on the displayed map.

== Installation ==

Installation of this plugin is fairly easy:

1. Unpack `ootinimaps-tooltips.zip`
2. Upload the whole directory and everything underneath to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
